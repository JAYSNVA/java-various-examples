rest-webapp-jersey-example
---------------------------

This is an example of using Spring IoC and JAX-RS/Jersey.

#### Technologies and software used:
* Java
* Spring
* JAX-RS/Jersey
* Tomcat
* Maven
* JUnit/Mockito
* Guava
* IntelliJ IDEA
